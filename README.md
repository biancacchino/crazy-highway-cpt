# Crazy Highway
Crazy Highway is a JavaFX 18 application created for my culminating task in highschool.
its an 80s themed shoot 'em up arcade themed game.

## Built with:
- Eclipse
- JavaFX 18

### Installation:
Simply download, extract, and run it in Eclipse.
